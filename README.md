# Experiment 29: Fibonacci Series

## Problem Statement

Given an integer $N$, find the $N^{th}$ number in the fibonacci series. Consider 0 and 1 to be the seed values.

**Series:** 0, 1, 1, 2, 3, 5...

## Input Format

* First line contains the integer $N$.

## Output Format

Print the $N^{th}$ fibonacci number.

### Example 1

**Input:**

```text
5
```

**Output:**

```text
3
```

### Explanation

The series starts 0, 1, 1, 2, 3...
* 1st: 0
* 2nd: 1
* 3rd: 1
* 4th: 2
* 5th: 3
